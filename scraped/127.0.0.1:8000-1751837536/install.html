Hosting This Site

This page contains instructions for installing the Apache 2 Web Server on various platforms and moving this site to the appropriate folder so that it can be viewed at the localhost address.

Before we start, please make sure you have a copy of the website. It is presumed that you do, since you are reading this! From this point onwards, the files for this site will be referred to as website.zip.

The instructions below will help you install Apache on Linux, OS X and Windows 7 platforms (other Windows versions should be similar).







Linux Installation

Open a terminal and run the following to update your repository:

sudo apt-get update

Then run the following to install Apache2:

sudo apt-get install apache2


The following instructions presume you have the website stored in your Downloads folder.

The document root for Apache is /var/www/. While in the terminal copy the website.zip file to this location and extract the contents:

sudo mv ~/Downloads/website.zip /var/www/

sudo unzip /var/www/website.zip

To check the installation, open your browser window and go to http://localhost/site1/index.html.





OS X Installation

Apache 2 is installed by default on Mac OS-X. To start the apache server, open a terminal and type the command:

sudo apachectl start


The following instructions presume you have the website stored in your Downloads folder.

The document root for Apache is Library/WebServer/Documents. While in the terminal copy the website.zip file to this location and extract the contents:

sudo mv ~/Downloads/website.zip /Library/WebServer/Documents

sudo unzip /Library/WebServer/Documents/website.zip

To check the installation, open your browser window and go to http://localhost/site1/index.html.





Windows Installation

Visit the Microsoft Download Centre to download the latest C++ Redistributable for Visual Studio 2015.

Make sure to select the correct version (32 or 64-bit) for your system.

Run the executable file and continue to the next step.

Visit the Apache Lounge and choose a suitable version of Apache for your Operating System (32 or 64-bit).
